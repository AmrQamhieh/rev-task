SECTION 1 — Linux / RHEL 9 Basics

Quiz A — Short Questions

What does setenforce 0 do on an SELinux-enabled system?

setting setenfore 0 puts SELinux into permissive mode, in this mode SElinux doesn't block any actions, it only gives warning logs of what would have been blocked if the SElinux is in Enforcing mode.

Explain the difference between /etc/hosts and DNS

/etc/hosts: it a local list of the hostnames and ip addresses localy DNS: it's a global distributed phone book for the internet, it converts domain names like www.google.com into an IP address

What is the purpose of /etc/fstab?

it's a configuration file that tells which filesystems to mount at boot and how to mount them. It lists things like:
  • disk partitions
  • network drivers
  • USB Drivers
  • Special filesystems like tmpfs

What is the difference between a service’s enabled and active state in systemd?

When systemd is enabled, this means that this service will be enabled after reboot automatically, Active in the otherhand, the system service is currently running but after reboot it is not.

How do you check which process is using a specific port (e.g., 8080)?

ss -tulpn | grep :8080 or netstat -tupln | grep :8080

What’s the difference between su and sudo?

su is for switching into another user, changing your current shell into another user's account sudo in the other, stands for "super user do", it lets you run one command with elevated privilleges, access is controlled by /etc/sudoers file

What’s the difference between firewalld zones and services?

Firewalld zones define the trust level for a network interface or source address range Services on the otherhand, they are set of rules representing a common network service. (e.g ssh)

What command shows the last 100 lines of a file and keeps following new lines live?

tail -n 100 -f path/to/file

What is the default target of systemd equivalent to runlevel 3?

multi-user.target

Quiz B — Practical Tasks

Create a user with a home directory, set a password, and add them to the wheel group.

  1. sudo useradd -m username
  2. sudo passwd username
  3. usermod -aG wheel username

Create a cron job that runs a backup script every day at 2 AM.

  1. crontab -e
  2. * 2 * * * * /path/to/backup.sh >> /var/log/backup.log 2>&1
  3. chmod +x /path/to/backup.sh

Find and delete all files larger than 100MB under /var/log.

sudo find /var/log -type f -size +100 -delete

Extend an LVM logical volume by 2GB and grow the filesystem.

sudo lvextend -L -r +2GB /dev/vgname/lvname

Block outgoing connections to port 3306 using firewalld.

sudo firewall-cmd --permanent --add-rich-rule='rule family="ipv4" port port="3306" protocol="tcp" direction="out" reject' firewall-cmd --reload
